# Ames Housing Data

This folder contains the Ames Housing dataset and its various processed versions used for regression analysis.

## Contents

- [Ames_Housing_Data.csv](Ames_Housing_Data.csv): The original dataset.
- [Ames_Housing_Feature_Description.txt](Ames_Housing_Feature_Description.txt): Description of the features in the dataset.
- [Ames_final.csv](Ames_final.csv): The final processed version of the data.
- [Ames_missing_removed.csv](Ames_missing_removed.csv): Version with missing values handled.
- [Ames_outliers_removed.csv](Ames_outliers_removed.csv): Version with outliers removed.
