# Ames Housing Data

This folder contains the Ames Housing dataset and its various processed versions used for regression analysis.

## Contents

- [Ames_Housing_Data.csv](file:///c:/Python/VSCPython/Models/data/ames/Ames_Housing_Data.csv): The original dataset.
- [Ames_Housing_Feature_Description.txt](file:///c:/Python/VSCPython/Models/data/ames/Ames_Housing_Feature_Description.txt): Description of the features in the dataset.
- [Ames_final.csv](file:///c:/Python/VSCPython/Models/data/ames/Ames_final.csv): The final processed version of the data.
- [Ames_missing_removed.csv](file:///c:/Python/VSCPython/Models/data/ames/Ames_missing_removed.csv): Version with missing values handled.
- [Ames_outliers_removed.csv](file:///c:/Python/VSCPython/Models/data/ames/Ames_outliers_removed.csv): Version with outliers removed.
