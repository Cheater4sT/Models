# Regression

This directory contains various implementations of regression models, organized by type and complexity.

## Subdirectories

- [Linear Regression](file:///c:/Python/VSCPython/Models/Regression/Linear%20Regression): Implementations using scikit-learn.
- [Polynomial Regression](file:///c:/Python/VSCPython/Models/Regression/Polynomial%20Regression): Non-linear regression techniques.
- [Regularization](file:///c:/Python/VSCPython/Models/Regression/Regularization): L1, L2, and Elastic Net regularization methods.
- [Simple Linear Regression](file:///c:/Python/VSCPython/Models/Regression/Simple%20Linear%20Regression): Basic single-variable linear regression.
