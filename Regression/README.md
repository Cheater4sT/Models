# Regression

This directory contains various implementations of regression models, organized by type and complexity.

## Subdirectories

- [Linear Regression](Linear%20Regression/): Implementations using scikit-learn.
- [Polynomial Regression](Polynomial%20Regression/): Non-linear regression techniques.
- [Regularization](Regularization/): L1, L2, and Elastic Net regularization methods.
- [Simple Linear Regression](Simple%20Linear%20Regression/): Basic single-variable linear regression.
