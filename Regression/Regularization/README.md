# Regularization

This directory covers techniques to prevent overfitting using regularization.

## Contents

- [l1_lasso.ipynb](l1_lasso.ipynb): Lasso (L1) regularization.
- [l2_ridge.ipynb](l2_ridge.ipynb): Ridge (L2) regularization.
- [l1l2_elastic_net.ipynb](l1l2_elastic_net.ipynb): Elastic Net regularization (combining L1 and L2).
